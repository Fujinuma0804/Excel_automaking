# Excel集計レポート生成ツール

「取得データ」「配点」「aaa（ひな型）」シートから受講者ごとの集計レポートを自動生成するツールです。

## 機能

- **総合得点シート**: セクション別の得点と総合得点を集計
- **5点評価シート**: セクション別と総合の5点評価を計算
- **個別レポートシート**: 各受講者ごとのレポートシートを自動生成

## 必要な環境

- Python 3.7以上
- openpyxl 3.1.0以上

## インストール

```bash
pip install -r requirements.txt
```

## 使い方

### GUI版（推奨）

```bash
python excel_report_generator.py
```

1. 「ファイルを選択」ボタンをクリックしてExcelファイル（.xlsm）を選択
2. 「レポートを生成」ボタンをクリック
3. 出力ファイルが同じフォルダに「_出力」を付けて保存されます

### Excelファイルの構造要件

- **「取得データ」シート**: 
  - L列（12列目）: 氏名
  - M列（13列目）: メールアドレス
  - Q列（17列目）から: 回答データ（0=不正解、1=正解）

- **「配点」シート**:
  - B列: セクション名
  - C列: 設問文
  - D列: 問題番号
  - E列: 配点

- **「aaa（ひな型）」シート**: 個別レポートのテンプレート

## 出力内容

1. **総合得点シート**: セクション別の得点と総合得点
2. **5点評価シート**: セクション別と総合の5点評価
3. **個別レポートシート**: 各受講者ごとのレポート（「{氏名}_レポート」という名前で作成）

## 注意事項

- 「配点」シートは毎回手動で作成する必要があります（自動化対象外）
- 各受講者レポート内の「総合的なコメント」「強みのコメント」「改善点のコメント」は自動化対象外です（AIで別途生成）
- 問題数やセクション数は試験によって異なりますが、自動的に対応します
